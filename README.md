# FRANK Cabal

Official page: **[frank.rh1.tech](https://frank.rh1.tech/)** — hub for all FRANK boards and firmware.

ScummVM port for RP2350-based FRANK boards, derived from [Cabal](https://github.com/project-cabal/cabal) — a community-maintained fork of [ScummVM](https://www.scummvm.org/).

Targets the Raspberry Pi Pico 2 (RP2350) with HDMI output, SD card storage, and PS/2 or USB input, enabling classic adventure games to run directly on the microcontroller.
